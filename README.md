## Simple Python Chatbot

A beginner-friendly Python chatbot built using conditional statements, loops, and dictionaries.
This project prints responses based on user input and continues the conversation until the user types "bye".

## Features

Greets the user
Answers basic questions
Supports simple conversational flow
Ends the chat when user types "bye"
Easy to understand and modify for beginners

## Project Structure
chatbox.py     # Main chatbot program
Readme.md      

## How It Works

The chatbot works by:
Taking user input
Converting it into lowercase
Checking if the input exists inside a dictionary of predefined responses
Returning an appropriate message
Repeating the loop until user types "bye"

## How to Run

python chatbox.py
